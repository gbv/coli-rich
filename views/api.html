Die auf Normdaten-Mappings basierende automatische Anreicherung von PICA-Datensätzen kann über eine API mit folgenden Endpunkten abgerufen werden.

GET /voc

Liefert die unterstützen Vokabulare als JSON-Array von JSKOS Concept Schemes.

GET /databases

Liefert die unterstützen Datenbanken im JSON-LD Format des GBV Datenbankverzeichnis.

GET /indexing

Liefert die in einem PICA-Datensatz enthaltenen Normdaten der unterstützen Vokabulare. URL-Parameter:

id={dbkey}:ppn:{ppn}

Auswahl des PICA-Datensatz per Datenbank-Kürzel und PPN

format={format}

picajson in PICA JSON
pp in PICA Plain
json in der intern von coli-rich verwendeten JSON-Struktur

GET /enrich

Liefert auf Grundlage von Normdaten-Mappings ermittelte Anreicherung von PICA-Datensätzen. URL-Parameter:

id={dbkey}:ppn:{ppn}

Auswahl des PICA-Datensatz per Datenbank-Kürzel und PPN
